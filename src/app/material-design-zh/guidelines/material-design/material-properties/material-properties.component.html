Material properties

Material的属性

Material has certain immutable characteristics and inherent behaviors.

Material有特定的属性和继承性的行为

Understanding these qualities of material will help you manipulate material in a way that’s consistent with the vision of material design.

理解这些material的特质，有助于你按照material design设想的方式来构成不同的material对象。
Material characteristics
Material的特质
Solid 立体的
Occupies unique points in space 在空间中的位置唯一
Impenetrable 不可透视
Mutable shape 形状可变
Changes in size only along its plane 仅在自身的平面上改变大小
Unbendable 不可弯曲
Can join to other material 能够合成到其他的material对象中
Can separate, split, and heal 可以分离、拆分和恢复
Can be created or destroyed 能够创建和删除
Moves along any axis 在任意轴上运动

Physical properties

物理特性

Material has varying x & y dimensions (measured in dp) and a uniform thickness (1dp).

Material具有不同的x、y轴线（单位为dp）和一个唯一的厚度（1dp）

 Do. 建议的做法 The height and width of material can vary. Material对象的长度和宽度可以不同 不建议的做法 Material is always 1dp thick. Material对象永远只有1dp的厚度

Material casts shadows.

Material对象产生阴影

Shadows result naturally from the relative elevation (z-position) between material elements.

阴影根据相对其他material对象的高度（z轴的位置）生成。

 Do. 建议的操作方式 Shadows depict the relative elevation between material elements. 阴影刻画了在不同material元素之间的相对高度 Don't. 不建议的操作方式 Shadows are never approximated by coloring material. 阴影永远不要采用与material元素相近的颜色。

Content is displayed on material, in any shape and color. Content does not add thickness to material.

内容在material对象上显示，能够具有任意的形状和颜色。但是内容并不增加material对象的厚度。

 Do. 建议的操作方式 Material can display any shape and color. Material对象可以显示任意的形状和颜色。

Content can behave independently of the material, but is limited within the bounds of the material.

内容能够在material对象上具备独立的行为，但是仅仅限于在material对象的边界范围内。

 Do. 建议的操作方式 Content behavior can be independent of the behavior of material. 内容在material对象上的行为是独立的。

Material is solid.

Material对象是不能穿透的。

Input events cannot pass through material.

输入事件不能穿透material对象。

 Do. 建议的操作方式 Input events only affect the foreground material. 输入时间进近印象前景的material对象。 Don't. 不建议的操作方式 Input events cannot pass through material. 输入时间不能穿透material元素

Multiple material elements cannot occupy the same point in space simultaneously.

多个material元素不能同时在控件中占用同样的位置点

 Do. 建议的操作方式

Using elevation to separate material elements is one method of preventing multiple material elements from occupying the same point in space simultaneously.

使用统一的高度划分方式来区分不同的material元素，防止多个material元素在空间中同时占用一个点

 Don't. 不建议的操作方式 Multiple material elements cannot occupy the same point in space simultaneously. 多个material元素不能同时在空间中占用同一个位置点。
